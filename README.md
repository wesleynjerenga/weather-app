# Weather App

This is a simple weather application made by me to show the weather of different areas.

## Features

- Displays current weather for any location
- Provides weather forecast for the next few days
- User-friendly interface

## Technologies Used

- **JavaScript**: 49.7%
- **CSS**: 36.5%
- **HTML**: 13.8%
